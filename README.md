hall_analyzerはパチスロ分析サイトのスロレポ様のデータを収集、分析するツールです
作成者もスロレポ様に問い合わせ、高負荷にならない程度のスクレイピングの許可をいただいています
スロットにハマった友人のために開発し、運用を継続しています
日付が変更されると自動でデータ取得＆分析されます
